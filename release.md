# KPomo v1.0.0 – Pomodoro Timer for Kindle

**Note:** This is the first public release of KPomo. If you have an older Kindle (5.6.1.1), download the zip with `-legacy` in its name.


### Features
- **25/5 Pomodoro cycle** with configurable durations
- Clean, Kindle‑friendly UI designed for black & white e‑ink screens
- Progress bar visualization of remaining time
- Start/pause/reset controls and simple keyboard input support
- Settings panel hides automatically during active timer
- +/- buttons for adjusting durations (no native spinners shown)
- Persistent state kept only for the current session (no storage required)

### Installation
1. Download the latest release archive (or build from source).
2. Connect your Kindle via USB and open the **Documents** folder.
3. Copy the entire `kpomo` directory and the `kpomo.sh` installer script.
4. Safely eject the device and open the **KPomo** booklet to launch.

### Notes for Users
- Designed to run in the Kindle browser; no external dependencies.
- Works on both legacy and newer Kindle models.
- Adjust the work/break lengths by tapping the numbers or using the +/– buttons.
